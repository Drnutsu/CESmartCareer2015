กลุ่มบริษัทซอฟต์สแควร์ ได้ก่อตั้งในปี พ.ศ. 2531 จากการรวมตัวของกลุ่มผู้ถือหุ้นผู้มีประสบการณ์ในการผลิตซอฟต์แวร์โปรแกรมพัฒนาระบบงาน และ Implement ระบบคอมพิวเตอร์สำหรับธุรกิจ ในองค์กร มาเป็นเวลา 26 ปี โดยมีวัตถุประสงค์ เพื่อให้บริการด้านการจัดหาระบบคอมพิวเตอร์ และเทคโนโลยีสารสนเทศ เพื่อประโยชน์สูงสุดขององค์กรธุรกิจ

โดยการทำหน้าที่เป็นศูนย์คอมพิวเตอร์ชั่วคราว บริษัทพร้อมที่จะให้คำแนะนำ และจัดหา Solution ที่เหมาะสมสำหรับแต่ละธุรกิจ ทั้งด้าน Hardware, System Software และ Application Software โดยมุ่งเน้นการผลิตการออกแบบและพัฒนา Application Software ให้มีความเหมาะสมตรงตามความต้องการของลูกค้า สามารถแก้ไขปัญหาและเพิ่มประสิทธิภาพในการดำเนินการ รวมถึงออกแบบให้ใช้งานง่าย, ใช้เวลาในการพัฒนาสั้น และทันกับความต้องการของธุรกิจ
