บริษัทของเราเป็นบริษัทเล็กๆ ที่กำลังเติบโตอย่างรวดเร็ว และเราอยากให้คุณเป็นส่วนหนึ่งของเราครับ  พวกเราทำ ธุรกิจเกี่ยวกับการสร้างโครงสร้างทางด้านเทคโนโลยี่สารสนเทศทั้งด้านสินค้าและการบริการ ให้กับองค์การทั้งในและต่าง ประเทศ  โดยมีหลักการในการนำเทคโนโลยี่ใหม่ล่าสุดที่เหมาะสมนำมาใช้ตอบสนองความต้องการที่ใช่  โดยงานที่พวกเรา เชี่ยวชาญรวมไปถีงโครงสร้างระบบเครือข่ายที่มี Performance สูง ระบบรักษาความปลอดภัยแบบควบคุมการเข้าถีงได้ใน องค์กร และระบบจำลองเสมือน Virtualization ทั้งด้าน Processing Power และ Storage สำหรับ ระบบ Application และ งานระบบต่างๆ

นอกเหนือจากงานด้านโครงสร้างสารสนเทศแล้ว  เนื่องจากบริษัทเรามีรากฐานมาจาก Silicon Valley เราก็ยังมีการ พัฒนาเทคโนโลยี่ Cloud ของตัวเองทั้งในด้าน Mobile และ Web application ครับ
