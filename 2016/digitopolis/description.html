บริษัท ดิจิโทโปลิส จำกัด เป็นบริษัทที่มุ่งเน้นในการพัฒนา Mobile Application และ Mobile Game ครับ โดยเรามีตำแหน่งงานทั้งในทั้งสองส่วน บริษัทมุ่งเน้นที่จะสร้างทีมงานที่มีความเชี่ยวชาญเฉพาะเพื่อให้ทีมงานมีทักษะที่เฉพาะในกระบวนการการสร้างสรรค์ผลิตภัณฑ์ นอกจากจะได้รับประสบการณ์ด้านการพัฒนาซอฟต์แวร์แล้ว ผู้ร่วมงานจะได้รับประสบการณ์ในการทำงานร่วมกับมืออาชีพจากสาขาอื่นๆ เช่น นักออกแบบเกม นักออกแบบกราฟฟิค และ นักทดสอบ เป็นต้น ดังนั้นจึงเหมาะมากสำหรับผู้สมัครที่ต้องการเรียนรู้เประสบการณ์ใหม่ๆ เกี่ยวกับการเขียนโปรแกรม และ การพัฒนาเกมบนมือถือ     สำหรับผู้ที่สนใจสามารถยื่นใบสมัครได้ในตำแหน่งงานต่อไปนี้ครับ
 
1. Junior Android Developer  จำนวน 1 ตำแหน่ง 
2. Senior Android Developer ( ประสบการณ์การพัฒนาแอปฯ 2 ปี ขึ้นไป) จำนวน 1 ตำแหน่ง 
3. Junior iOS Developer    จำนวน 2 ตำแหน่ง 
4. Junior Server End Developer  จำนวน 1 ตำแหน่ง
5. Game Server Developer จำนวน 2 ตำแหน่ง
6. UNITY Game Developer จำนวน 1 ตำแหน่ง 
7. User Experience Designer, User Interface Designer จำนวน 1 ตำแหน่ง
