ก่อนจะมาเป็น BUILK

เรามีความคิดที่ว่า เทคโนโลยีน่าจะช่วยให้ทำงานก่อสร้างให้เสร็จตามงบประมาณได้ เลยชวนพรรคพวกสายเทคโนโลยีมาทำโปรแกรมควบคุมต้นทุนงานก่อสร้างกัน พอแก้ปัญหาต้นทุนให้ตัวเองได้แล้ว ก็อยากจะแบ่งปันเพื่อนร่วมวงการก่อสร้างให้มีระบบคุมต้นทุนบ้าง

BUILK.COM จึงเกิดขึ้นมาเมื่อปี 2553 ให้ผู้รับเหมาใช้คุมต้นทุนออนไลน์แบบมืออาชีพกันฟรีๆ ไม่มีกั๊ก แล้วหารายได้จากค่าโฆษณาและงบการตลาด ของแบรนด์วัสดุก่อสร้างแทน

ใน 5 ปีที่ผ่านมา เราได้รางวัลโปรแกรมระดับประเทศ รวมถึงระดับภูมิภาคมามากจนโด่งดังในวงการไอทีสตาร์ทอัพ ส่วนสมาชิก ผู้รับเหมา ก็เพิ่มขึ้นเรื่อยๆ ทั้งไทย ลาว อินโดนีเซีย มาเลเซีย จนเราต้องขยายพื้นที่เก็บข้อมูลจากไปใช้ Cloud Service ของ AWS และเพิ่มโปรแกรมเมอร์มาปรับปรุง อัพเดท เพิ่มความปลอดภัยระบบให้กับผู้ใช้โปรแกรมมาตลอด

แต่ไปๆมาๆ เมื่อผู้ใช้งานเพิ่มมากขึ้น ค่าโฆษณาอย่างเดียวคงไม่พอกับค่าพัฒนารักษาโปรแกรมที่เพิ่มขึ้นปีละหลายล้านบาท เราก็ขยับโมเดลธุรกิจเข้าสู่ธุรกิจ e-Commerce สำหรับวงการก่อสร้าง

ทุกวันนี้ Builk พัฒนา application เพื่อคนก่อสร้างโดยอาศัยน้ำพักน้ำแรงจากทุกคนในทีมทำงานร่วมกันภายใต้ระบบเดียวกัน องค์กรแบบ flat เปิดโอกาสให้คิด ถาม เสนอไอเดียได้อิสระในที่ประชุม

 

“Agile เบาๆ แต่เราทำงานจริงจัง!”

 

ด้วยความที่เป็นบริษัทขนาดเล็ก เราจึงต้องการทีมงานที่มีใจรัก รับผิดชอบต่อหน้าที่ และมีความสามารถล้นเหลือ Multi-talented พึงระลึกไว้เสมอว่า...

 

"ทำงานที่นี่ ไม่ได้มีแต่ coding"

 

วันดีคืนดีคุณอาจได้ช่วยจัดอีเวนท์ ได้พบปะ user ที่ใช้โปรแกรมของคุณตัวเป็นๆ เพราะงั้น communication ก็เป็นสกิลที่ขาดไม่ได้

 

ออฟฟิศบิลค์ตั้งอยู่ใจกลางถนนทรัพย์ เดินทางสะดวกทั้ง BTS และ MRT ใกล้สามย่าน อาหารการกินไม่ขาดด้วยสวัสดิการขนมประจำออฟฟิศทุกอาทิตย์ และเสื้อยืด Builk สีเหลืองสดใสที่จำกัดสิทธิ์ไว้สำหรับพนักงาน (และแฟนคลับตัวจริงของ Builk) เท่านั้น

::Frontend Developer:: (1 position)

Frontend Developer skill set

 

DTO Serialization : JSON

ต้องรู้ ว่า JSON คืออะไร ใช้ยังไง แกะได้ ประกอบได้ เอาไปใช้ได้

Frontend Programming language : HTML and Javascript and CSS

::Special Bonus for Frontend dev::

Backend Programming language : C# or Python or whatever

UX
